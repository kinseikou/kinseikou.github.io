中国親戚計算機ツール

現在の親戚呼称大全には約-の親戚関係が収録されている。

親族は結婚、血縁、法律制定に基づいて形成された社会関係である。中国の親族関係には、夫妻、父母、子女、兄弟姊妹、祖父母と外祖父母、孙子女と外孙子女、儿媳と公婆、女婿と岳父母、おじ、伯、叔、姑、舅、姨、侄子女、甥子女、堂兄弟姊妹、表兄弟姊妹などの三世代以内の傍系血族が含まれている。

親族呼称とは、本人を中心に親族のメンバーと本人の関係を特定する名称であり、血親縁故に基づく親族間の相互呼称の名称、呼び方である。漢族の家族親族関係は、自分より一世代上の呼び方には叔父叔父叔父、同世代には兄弟姉妹、いとこなどがあり、次の世代には甥姪などがある。中国人の縁故による親戚関係はかなり複雑で、「祖先十八代」、「五服」、「六親」、「九族」の区別がつきにくく、派生した各種呼称も中華文化の大きな特徴である。

「中国親戚関係計算機」には中国の親戚関係の呼称大全が収録されており、簡単な入力だけで呼称計算が完了し、中国式の親戚関係の換算を簡単に把握でき、中国文化をより理解できるようになっています。

機能サポート

  • 1.別名クエリの使用：

    姥姥の爸爸の老窦 = 外曾外曾祖父

    娘子の爹地の母亲 = 祖岳母

    岳丈の妈咪の细佬 = 舅祖岳父

  • 2.合称クエリを使用する

    姐夫の双亲 = 姊妹姻父 / 姊妹姻母

    妈妈の祖父母 = 外曾祖父 / 外曾祖母

    弟弟の子 = 侄子 / 侄女

  • 3.サイズ番号混合クエリー：

    大哥の二姑妈の七舅姥爷 = 舅曾外祖父

    二女婿の小姨 = 女姻姨妹

    大佬の三叔の次子 = 堂哥 / 堂弟

  • 4.先祖孫のスパンクエリを制限しない：

    舅妈の婆婆の外甥の姨妈の侄子 = 舅表舅父

    老公の丈母娘の儿媳 = 嫂子 / 弟妹

    外婆の孙子の爸爸の仔 = 舅表哥 / 舅表弟

  • 5.年齢による可能性の推定：

    哥哥の表姐 = 姑表姐 / 舅表姐

    堂妹の姐姐 = 堂姐 / 堂妹

    弟弟の表哥 = 姑表哥 / 姑表弟 / 舅表哥 / 舅表弟

  • 6.文脈に基づいて性別を確認する：

    老婆の女儿の外婆 = 岳母

    爱人の婆婆の儿子 = 哥哥 / 弟弟 / 自分

    岳母の配偶の儿子の爸爸 = 岳父

  • 7.漢文式表現を支持する：

    吾父之舅父 = 舅爷爷

    母之兄长之妻 = 大舅妈

    泰山之高堂 = 祖岳父 / 祖岳母

  • 8.ある呼称関係チェーンを解析する：

    七舅姥爷 = 妈妈の妈妈の兄弟

    姑奶奶 = 爸爸の爸爸の姐妹

    外太公 = 妈妈の爸爸の爸爸

  • 9.二人の親戚間の合称関係を計算する：

    奶奶 + 外婆 = 儿女亲家

    妈妈 + 婶婶 = 妯娌

    岳父 + 私 = 翁婿
